# Loan Approval Analysis and Prediction

## Project Overview

This project focuses on analyzing loan approval data to understand the factors that influence loan approval decisions. The dataset is cleaned, processed, and explored using Python data analysis libraries. Various visualization techniques are applied to identify patterns and relationships between applicant attributes and loan approval status.

The project demonstrates the complete data analysis workflow, including data exploration, preprocessing, feature engineering, and visualization. The processed dataset can also be used for future machine learning applications.

---

## Problem Statement

Banks and financial institutions receive numerous loan applications and must determine whether an applicant qualifies for a loan. Making these decisions manually can be time-consuming and inefficient.

This project aims to analyze loan applicant data and identify the key factors that affect loan approval decisions through data preprocessing and exploratory data analysis techniques.

---

## Objectives

- Understand the structure of the loan approval dataset.
- Perform Exploratory Data Analysis (EDA).
- Identify and handle missing values.
- Clean and preprocess the dataset.
- Convert categorical variables into numerical format.
- Analyze relationships between different features.
- Visualize important trends and patterns.
- Prepare the dataset for predictive modeling.

---

## Dataset Description

The dataset contains information related to loan applicants and their approval status.

### Features Included

- Age
- Income
- Credit Score
- Employment Status
- Loan Amount
- City
- Loan Approval Status

The target variable is **Loan Approval Status**, which indicates whether a loan application is approved or rejected.
